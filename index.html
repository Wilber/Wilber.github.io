RNA-Seq data analysis at OSC, 2021

Wilbur Ouma, Senior Systems Consultant (Client Engineer)

Outline

In this workshop, you will learn how to analyze RNA-seq count data, using R. This will include reading count data into R, quality control, and performing differential expression analysis, with a focus on the edgeR package. This workshop is aimed at biologists interested in learning how to perform differential expression analysis of RNA-seq count data. Preprocessing of raw sequence reads will not be covered here.

Course Prerequisites & Requirements

Some basic R knowledge is assumed. If you are not familiar with the R statistical programming language we strongly encourage you to work through the Intro to R materials before attempting these materials. In addition, an understanding of basic probability and statistics concepts — such as probability distributions and linear regression models — is encouraged.

Learning Objectives:

  • To pre-process RNA-Seq read counts
  • To identify deferentially expressed genes (DEGs)

Data

Data for this workshop comes from a Nature Scientific Data paper by Zhang et al., 2018. The corresponding count data is available in the figshare repository: https://ndownloader.figshare.com/files/23241812

Software

You will be learning how to analyze RNA-seq count data, using R, with focus on the edgeR package. Additional R packages that will be used include: tidyverse (for data wrangling, includes ggplot2, dplyr, tidy, readr); RColorBrewer (a colour scheme for plotting); and Glimma (for interactive plots). If you are not familiar with installing and loading R packages, see section 0.4 of Intro to R. We will use the R environment at the Ohio Supercomouter Center (OSC).
