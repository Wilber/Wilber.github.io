Multi-factor RNA-Seq Data Analysis

0.1 Model fitting and Hypothesis testing

0.1.1 Method 2: Accounting for location

Recall that from Method 1, we fit two separate models on data from the two locations, and identifies DEGs due to application of salt separately. If the biological question is to identify the effect of treatment while the location is held contstant (i.e the effect of treatment irrespective of the location), then we need to account for location. We observed from MDSplot from Day 1 that there’s some (biological) variation explained by location due to separation of samples by Dimension 2. We therefore need to fit a multi-factor (multi-variate) additive model that accounts for location as a covariate.

Let us first load required packages:

library(edgeR) #Main package for DE analysis
library(tidyverse) #Data wrangling package, includes ggplot2, dplyr, tidy, readr
library(RColorBrewer) #Colour scheme for plotting
library(Glimma) #Interactive MD plots
library(gplots)

Then get the original copy of the DGE object

dge_method2<-readRDS("dge_orig.rds")

Note that for a multivariate model, all combinations of treatments/group should be replicated in the second variable/factor. That is, we should have Control (C) and Treatment (T) samples in both inland and beach. We specify the group factor in the dge_method2 to reflect this:

group<-as.factor(rep(c("C","T","C","T"), c(3,3,3,3)))
dge_method2$samples$group<-group
#verify that we have "C" and "T" in both inland and beach
dge_method2$samples

0.1.1.1 Data preprocessing

First, let us preprocess the count data contained in the dge_methods2 object

  # perform cpm and lcp transformation:
  cpm <- cpm(dge_method2)
  lcpm <- cpm(dge_method2, log=TRUE)
  #filter low expressed genes
  keep.exprs <- filterByExpr(dge_method2)
  dge_method2 <- dge_method2[keep.exprs,, keep.lib.sizes=FALSE]
  ##TMM normalization:
  dge_method2 <- calcNormFactors(dge_method2, method = "TMM")

0.1.1.2 Design matrix

Let us create a design matrix, this time round with intercept for the location variable. Also,this type of additive model corrects/adjusts for baseline differences between the locations. IMPORTANT: The order of factors matters—always put the factor of interest last!

location <- dge_method2$samples$location    
mod <- model.matrix(~ location + group)
rownames(mod) <- colnames(dge_method2)

We see three coefficients (sometimes called “dummy variables”), the last coefficient (coefficient 3) representing the effect of treatment on gene expression, after adjusting for the location. This is our coefficient of interest.

Next we estimate dispersion:

  dge_method2 <- estimateDisp(dge_method2, design = mod)

We fit a GLM

  fit <- glmFit(dge2, mod)

We perform hypothesis testing to answer the following biological question: * What genes are differentally expressed under treatment, after accounting for location? Note below that we do not build a contrast matrix, but rather we specify the coefficient of interest (3) while performing the LRT.

#LRT
glf_T_vs_C <-glmLRT(fit, coef = 3)

#perform multiple test correction, and obtain DEGs with FDR cutfoff of 0.1
pvals_T_vs_C <- topTags(glf_T_vs_C, n = "Inf", adjust.method = "BH", sort.by = "PValue", p.value = 0.1)[[1]]
